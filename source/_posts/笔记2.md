## 数据处理
### 浮点数2  

>浮点数和整数比较可以计算中间值，表示范围较大，但是运算速度会慢，同时精度降低。

```cpp
#include <iostream>
int main()
{
using namespace std;
float a = 2.34E+22f;
float b = a + 1.0f;
cout << "a ="<< a <<endl;
cout << "b - a ="<< b - a  <<endl;
    return 0;
}
```
```text
a =2.34e+22
b - a =0
```
>b和a之间差了1，但是运算结果确实0，是因为上一节讲到float十进制有效位是6-7位，2.34×10的22次方是一个远超7位的数字，以下详细演示：
```cpp
#include <iostream>
int main(void)
{
    using namespace std;

    float hats , heads;
    cout.setf(ios_base::fixed,ios_base::floatfield);
    cout << "enter a number :";
    cin >>hats;
    cout << "enter a number :";
    cin>>heads;
    cout <<"hats="<<hats<<":heads"<<heads <<endl;
    out << "hats+heads=" << hats+heads << endl;
    return 0;
    }
```

```text
enter a number :23.34
enter a number :34.56
hats=23.340000:heads34.560001
hats+heads=57.900002
```
>小数点后显示6位是因为在cout.setf(ios_base::fixed,ios_base::floatfield);代码中要求了定点显示。

#### 除法分支
```cpp
#include <iostream>
int main(void)
{
    using namespace std;
    cout.setf(ios_base::fixed, ios_base::floatfield);
    cout << " chufa 9/5=" << 9/5 <<endl;
    cout << " chufa 9.0/5.0=" << 9.0 / 5.0 << endl;
    cout << " chufa 9.0/5=" << 9.0 / 5 << endl;
    cout << " doubie 1e7/9.0=" << 1e7 / 9.0 << endl;
    return 0;
}
```

```text
 chufa 9/5=1
 chufa 9.0/5.0=1.800000
 chufa 9.0/5=1.800000
  chufa 9/5=1
 chufa 9.0/5.0=1.800000
 chufa 9.0/5=1.800000
 doubie 1e7/9.0=1111111.111111
 ```

- 以上可以看出如果两个操作数都是整数，则执行整数除法舍弃小数位，如果其中有一个数为浮点值结果为浮点数。
